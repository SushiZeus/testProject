## 🎯 Quick Test Workflow

1. **Open File** (Documentation Officer)
   - Login → File Opening → Create new file

2. **Assign Declarant** (Declaration Manager)
   - Login → Declaration → Assign to declarant

3. **Process Declaration** (Declarant)
   - Login → Declaration → Upload tax & wharfage
   - Click TAX PAID and WHARFAGE PAID

4. **Operations** (Operation Clerk)
   - Login → Operations → Process file
   - Upload port charges
   - Click PORT CHARGES PAID

5. **View Timeline** (Any Role)
   - Click on file → View activity timeline
   - See user names and actions
