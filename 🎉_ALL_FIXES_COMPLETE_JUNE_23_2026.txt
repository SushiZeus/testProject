📊 WORKFLOW SUMMARY
───────────────────────────────────────────────────────────────────

DECLARATION WORKFLOW:
  1. Upload tax documents → Status: WAITING FOR TAX PAYMENT
  2. Click TAX PAID (Blue) → Sets taxPaymentConfirmed flag
  3. [SEA ONLY] Upload wharfage → Status: WAITING FOR WHARFAGE PAYMENT
  4. [SEA ONLY] Click WHARFAGE PAID (Blue) → Sets wharfagePaymentConfirmed
  5. Click DECLARATION DONE (Green) → Status: READY FOR OPERATIONS

OPERATIONS WORKFLOW (SEA):
  1. Upload release order → Status: RELEASE ORDER RECEIVED
  2. Upload port charges → Status: WAITING FOR PORT CHARGES PAYMENT
  3. Finance clicks PORT CHARGES PAID (Blue) → Sets portChargesPaidAt
  4. Clerk clicks OPERATIONS DONE (Green) → Status: VERIFICATION FORM PENDING
  5. Fill Verification Form → Status: VERIFICATION FORM COMPLETED
  6. Click OPERATIONS DONE (Blue) → Status: DRIVER REQUESTED

OPERATIONS WORKFLOW (AIR):
  1. Upload release order → Status: RELEASE ORDER RECEIVED
  2. Upload swissport charges → Status: WAITING FOR SWISSPORT CHARGES PAYMENT
  3. Finance clicks SWISSPORT PAID (Blue) → Sets swissportChargesPaidAt
  4. Clerk clicks OPERATIONS DONE (Green) → Status: VERIFICATION FORM PENDING
  5. Fill Verification Form → Status: VERIFICATION FORM COMPLETED
  6. Click OPERATIONS DONE (Blue) → Status: DRIVER REQUESTED
