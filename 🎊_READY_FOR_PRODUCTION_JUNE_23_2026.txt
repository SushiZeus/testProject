🛠️ IF YOU ENCOUNTER ISSUES
═══════════════════════════════════════════════════════════════════════════════

SERVER NOT STARTING:
→ Check if port 5173 is already in use
→ Close other applications using the port
→ Try restarting computer

CHANGES NOT VISIBLE:
→ Clear browser cache (Ctrl + Shift + Delete)
→ Hard refresh page (Ctrl + F5)
→ Try different browser

BUTTON NOT WORKING:
→ Check you're logged in with correct role
→ Verify file is in correct status
→ Check if previous steps were completed
→ Refresh page and try again

NOTIFICATION NOT RECEIVED:
→ Check notification bell icon (top right)
→ Verify user ID is correct in system
→ Refresh page
→ Check if notification store is working
