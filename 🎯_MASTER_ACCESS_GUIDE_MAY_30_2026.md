## 🔄 COMPLETE WORKFLOW

### **1. FILE OPENING** (Documentation Officer)
1. Login to system
2. Navigate to "File Opening"
3. Select client (existing or new)
4. Choose shipment type (IMPORT/EXPORT)
5. Choose service type (CLEARANCE/DOCUMENT_HANDOVER/TRANSPORTATION)
6. Select transport mode (SEA/AIR/ROAD)
7. Upload required documents
8. Submit file
9. System assigns to declarant

### **2. DECLARATION** (Declarant)
1. Receive notification
2. Navigate to "Declaration" module
3. Acknowledge file
4. Process customs declaration
5. Upload tax assessment document
6. Upload tax payment document
7. Click "TAX PAID" ✅ (declarant confirms)
8. For SEA: Upload wharfage payment document
9. For SEA: Click "WHARFAGE PAID" ✅ (declarant confirms)
10. Click "DECLARATION DONE"
11. File moves to operations

### **3. OPERATIONS** (Operation Clerk)
1. Receive notification
2. Navigate to "Operations" module
3. Click "Accept" button
4. Upload verification photos (1-7 photos)
5. Upload release order
6. **For SEA**: Upload port charges
7. **For AIR**: Upload Swissport charges
8. Wait for Finance to confirm payment ⏳
9. After payment confirmed:
   - **SEA**: Click "OPERATIONS DONE"
   - **AIR**: Click "CARGO CLEARED"
10. File status → `VERIFICATION_FORM_PENDING`

### **4. CARGO VERIFICATION** ⭐ NEW (Operation Clerk)
1. File shows purple "Fill Verification Form" button
2. Click button to open form dialog
3. Fill all required fields:
   - Station (ICD)
   - Verification Date
   - Verification Officer name
   - TANSAD Number
   - Answer all 10 YES/NO questions
   - Add remarks for each question
   - Fill general remarks (optional)
   - Enter officer signature name
4. Submit form
5. File status → `VERIFICATION_FORM_COMPLETED`
6. File ready for driver assignment
7. Can view/print form anytime (green button)

### **5. DELIVERY** (Transport Manager & Driver)
1. Transport Manager assigns driver
2. Driver receives notification
3. Driver accepts job
4. Driver updates status: Collecting → Collected → Delivering → Delivered
5. File completed

## 📋 CARGO VERIFICATION FORM DETAILS

### **Form Header Fields:**
- **Station (ICD)**: Required - e.g., "ICD Dar es Salaam"
- **Verification Date**: Required - Auto-populated with current date
- **Verification Officer**: Required - TRA officer or company officer name
- **TANSAD Number**: Required - Customs system reference number

### **10 Inspection Questions** (All Required - YES/NO + Remarks):

1. **Physical damage to external part?**
   - Check for visible external damage
   - Add remarks if damage found

2. **Are seals still intact?**
   - Verify container/cargo seals
   - Note any broken or missing seals

3. **Discrepancy in cargo quantity/CHS no?**
   - Compare actual quantity with documents
   - Note any differences

4. **Same items as declared in customs entry?**
   - Verify cargo matches declaration
   - Document any discrepancies

5. **All parts seen (spare parts, logo)?**
   - Check for complete shipment
   - Note missing components

6. **Internal damage to goods?**
   - Inspect cargo condition
   - Document any internal damage

7. **Do goods need special attention (packaging)?**
   - Assess packaging requirements
   - Note special handling needs

8. **Was container locked after verification?**
   - Confirm container security
   - Document lock status

9. **Was seal fixed to container?**
   - Verify new seal applied
   - **Enter seal number** (required if YES)

10. **Samples taken by officer?**
    - Document if samples collected
    - **Enter quantity** (required if YES)

### **Form Footer:**
- **General Remarks**: Optional - Additional observations
- **Officer Name**: Required - Signature/name of officer completing form

### **Print Format:**
Professional table layout matching original PDF verification sheet, including:
- Company header
- File reference number
- All inspection questions with checkboxes
- Remarks sections
- Signature blocks
- Date fields
- Notes and instructions
