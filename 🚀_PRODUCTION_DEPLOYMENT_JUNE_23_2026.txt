🔄 COMPLETE WORKFLOW (END-TO-END)
═══════════════════════════════════════════════════════════════════════════════

STEP 1: FILE OPENING
────────────────────
User: Documentation Officer
Action: Create new file with client and shipment details
Status: WAITING_FOR_DECLARATION

STEP 2: DECLARATION ASSIGNMENT
──────────────────────────────
User: Declaration Manager
Action: Assign file to declarant
Status: ASSIGNED_TO_DECLARANT

STEP 3: DECLARATION PROCESSING
──────────────────────────────
User: Declarant
Actions:
  - Acknowledge file → WAITING_FOR_FINAL_ASSESSMENT
  - Upload tax documents → WAITING_FOR_TAX_PAYMENT
  - Click TAX PAID → taxPaymentConfirmed = true
  - [SEA] Upload wharfage → WAITING_FOR_WHARFAGE_PAYMENT
  - [SEA] Click WHARFAGE PAID → wharfagePaymentConfirmed = true
  - Click DECLARATION DONE → READY_FOR_OPERATIONS

STEP 4: OPERATIONS ASSIGNMENT
─────────────────────────────
User: Operations Manager
Action: Assign file to operation clerk
Status: RECEIVED_BY_CLERK

STEP 5: OPERATIONS PROCESSING
─────────────────────────────
User: Operation Clerk
Actions:
  - Upload release order → RELEASE_ORDER_RECEIVED
  - [SEA] Upload port charges → WAITING_FOR_PORT_CHARGES_PAYMENT
  - [SEA] Finance clicks PORT CHARGES PAID → portChargesPaidAt set
  - [AIR] Upload swissport → WAITING_FOR_SWISSPORT_CHARGES_PAYMENT
  - [AIR] Finance clicks SWISSPORT PAID → swissportChargesPaidAt set
  - Click OPERATIONS DONE → VERIFICATION_FORM_PENDING
  - Fill verification form → VERIFICATION_FORM_COMPLETED
  - Click OPERATIONS DONE → DRIVER_REQUESTED

STEP 6: DELIVERY REQUEST
────────────────────────
User: Delivery Clerk
Actions:
  - View file in Delivery Module
  - Click REQUEST DRIVER
  - Select SMALL or BIG truck
  - Notification sent to appropriate manager

STEP 7: DRIVER ASSIGNMENT
─────────────────────────
User: HR Manager (small) or Transport Manager (big)
Actions:
  - Receive notification
  - View file in driver module
  - Assign available driver
  - Driver receives notification

STEP 8: DELIVERY EXECUTION
──────────────────────────
User: Driver
Actions:
  - Accept assignment
  - Status: DRIVER_ASSIGNED
  - Collect cargo
  - Status: DRIVER_COLLECTING_CARGO
  - Cargo collected
  - Status: CARGO_COLLECTED_FROM_ICD/AIRPORT
  - Deliver to client
  - Status: DELIVERED_TO_CLIENT ✅
