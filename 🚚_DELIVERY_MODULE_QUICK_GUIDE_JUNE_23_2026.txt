🎯 COMPLETE WORKFLOW SUMMARY

STEP 1: OPERATIONS COMPLETE
────────────────────────────────────────────────────────────────────────────
User: Operation Clerk
Action: Click OPERATIONS DONE (after verification form)
Result: File moves to DRIVER_REQUESTED status
Alert: Delivery Clerk receives notification → File appears in Delivery Module


STEP 2: DELIVERY CLERK REQUESTS DRIVER
────────────────────────────────────────────────────────────────────────────
User: Delivery Clerk
Module: /delivery
Button: REQUEST DRIVER (Purple) - Click to open truck size dialog
Options: 
  🚙 Small Truck → Routes to HR Manager
  🚛 Big Truck  → Routes to Transport Manager


STEP 3: TRUCK SIZE SELECTED
────────────────────────────────────────────────────────────────────────────
Selection: Small Truck
├─ Notification: HR Manager (@hr_manager@company.com)
├─ Title: "Driver Request - SMALL Truck"
├─ Action: Go to /drivers-management to assign
└─ Drivers Available: HR's small truck drivers

Selection: Big Truck
├─ Notification: Transport Manager (@transport_manager@company.com)
├─ Title: "Driver Request - BIG Truck"
├─ Action: Go to /drivers to assign
└─ Drivers Available: Transport's big truck drivers


STEP 4: MANAGER ASSIGNS DRIVER
────────────────────────────────────────────────────────────────────────────
User: HR Manager (small) or Transport Manager (big)
Module: /drivers-management (HR) or /drivers (Transport)
Action: Select appropriate driver from available list
Result: Driver assigned to file


STEP 5: DRIVER RECEIVES NOTIFICATION
────────────────────────────────────────────────────────────────────────────
Notification: "Driver Assignment"
Message: "You have been assigned to file DOW-XXXXX"
Action: Accept/decline and proceed to collection point
Status Update: DRIVER_ASSIGNED → DRIVER_COLLECTING_CARGO


STEP 6: DELIVERY COMPLETE
────────────────────────────────────────────────────────────────────────────
Driver Actions:
  ✓ Collect cargo from ICD/Airport
  ✓ Transport to client
  ✓ Deliver and get signature
  ✓ Mark as complete

Final Status: DELIVERED_TO_CLIENT
